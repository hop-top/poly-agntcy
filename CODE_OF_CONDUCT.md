# Code of Conduct

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful and constructive. Harassment, personal attacks, and bad-faith engagement are not welcome. Maintainers may remove or block participants whose behavior violates this code.

## Enforcement

Report incidents to **me@jadb.io**. Reports are confidential. Response within 48 hours.

The full Contributor Covenant text — including scope, expected behavior, and enforcement responsibilities — applies as published.
