# Installing poly-agntcy

## From Source

```sh
git clone hop.top/agntcy/poly-agntcy
cd poly-agntcy
task build
```

## Go

```sh
go install hop.top/agntcy/poly-agntcy@latest
```

## npm

```sh
npm install -g poly-agntcy
```

## pip

```sh
pip install poly-agntcy
```

## Homebrew

```sh
brew install poly-agntcy
```

## Verify Installation

```sh
poly-agntcy --version
```

## System Requirements

- Go 1.22+ (for Go builds)
- Node.js 20+ (for TypeScript builds)
- Python 3.11+ (for Python builds)
